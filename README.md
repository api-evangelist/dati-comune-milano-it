# City of Milan Open Data (dati-comune-milano-it)

City of Milan Open Data is a open data portal open-data portal for Italy running **CKAN**, exposing a standardized catalog API over ~2,594 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/dati-comune-milano-it/refs/heads/main/apis.yml)
